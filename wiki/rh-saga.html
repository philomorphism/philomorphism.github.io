  sage: NumberField(x^2+1,"x").zeta_coefficients(20)
  [1, 1, 0, 1, 2, 0, 0, 1, 1, 2, 0, 0, 2, 0, 0, 1, 2, 1, 0, 2]
  
  sage: EllipticCurve("14.a5").anlist(20)
  [0, 1, -1, -2, 1, 0, 2, 1, -1, 1, 0, 0, -2, -4, -1, 0, 1, 6, -1, 2, 0]

where "14.a5" is defined on The L-functions and modular forms database. understanding this "algorithm" and the underlying pattern is an instance of Langland's Program!
